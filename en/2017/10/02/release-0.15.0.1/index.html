Bitcoin Core 0.15.0.1 Released

by .

Overview

  • Hashes for verification

Bitcoin Core 0.15.0.1 has been released with a fix for a minor bug in 0.15.0, which caused the GUI client to crash for some users after upgrading to 0.15.0.

After upgrade to 0.15.0, some clients would crash at startup because a custom fee setting was configured that no longer exists in the GUI. This is a minimal patch to avoid this issue from occurring.

Please see the release notes for details. To download, please visit the download page.

If have any questions, please stop by either our IRC or Slack chatrooms and we’ll do our best to help you.
