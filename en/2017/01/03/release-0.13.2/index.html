Bitcoin Core 0.13.2 Released

by .

We are please to announce the general availability of Bitcoin Core 0.13.2. This release forms part of the regular maintenance cycle of Bitcoin Core and brings bug fixes, optimisations and improvements to the 0.13.x series.

Please see the release notes for more details.

Download Now

Binaries are available from the download directory. Power users can fetch and compile from the Bitcoin Core v0.13.2 release tag. If downloading binaries, please ensure you check the verification hashes below and from the download page. If compiling from source, please check the v0.13.2 tag signature matches.
